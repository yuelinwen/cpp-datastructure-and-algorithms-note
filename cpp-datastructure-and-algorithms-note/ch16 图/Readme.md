ch16 图


图是一种用来表示**“多对多关系”**的数据结构。
图 = 顶点（Vertex） + 边（Edge）


有向图 vs 无向图
无向图	边无方向
有向图	边有方向


权图 vs 无权图
带权图：边有权值（距离 / 花费）
无权图：只有连接关系


度（Degree）
 无向图：结点连接的边数
有向图：
入度 din（In-degree）: 指的是 至 该顶点的边数
出度 dout（Out-degree）：指的是 关联于 该顶点的边数

图的遍历
 BFS（广度优先搜索）
用 队列 一层一层扩展 最短路径（无权图）
从一个顶点开始，搜索所有可达到顶点的方法
DFS（深度优先搜索）
用 递归 / 栈 一条路走到底 检测环 / 拓扑排序
