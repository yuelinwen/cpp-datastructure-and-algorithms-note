Queue（队列）操作时间复杂度
3️⃣ Array 实现（循环队列） 用数组 + front / rear 指针 + 取模
| 操作           | 时间复杂度    | 原因             |
| ------------ | -------- | -------------- |
| enqueue      | **O(1)** | 移动 `rear`（取模）  |
| dequeue      | **O(1)** | 移动 `front`（取模） |
| front / peek | **O(1)** | 直接访问           |
| isEmpty      | **O(1)** | 判断指针关系         |

4️⃣ LinkedList 实现（链队列）用头指针 + 尾指针
| 操作           | 时间复杂度    | 原因    |
| ------------ | -------- | ----- |
| enqueue      | **O(1)** | 尾插法   |
| dequeue      | **O(1)** | 删除头结点 |
| front / peek | **O(1)** | 访问头结点 |
| isEmpty      | **O(1)** | 判断头指针 |
