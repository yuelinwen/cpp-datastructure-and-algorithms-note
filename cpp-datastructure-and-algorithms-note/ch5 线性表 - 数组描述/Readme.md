Arraylist 具体实现
| 操作                  | 最好情况        | 最坏情况 | 说明              |
| ------------------- | ----------- | ---- | --------------- |
| 访问元素 `get(i)`       | O(1)        | O(1) | 直接用下标访问         |
| 修改元素 `set(i, x)`    | O(1)        | O(1) | 直接用下标修改         |
| 插入元素 `insert(i, x)` | O(1)（末尾）    | O(n) | 如果在中间，需要移动后面的元素 |
| 删除元素 `delete(i)`    | O(1)（末尾）    | O(n) | 删除中间元素需要移动后面的元素 |
| 查找元素 `find(x)`      | O(1)（找到第一个） | O(n) | 顺序查找            |
| 长度 `length()`       | O(1)        | O(1) | 一般用变量存储         |



vector
C++ 普通数组大小固定、管理手动、功能简单；vector 是动态数组，支持自动扩容、丰富操作接口、自动管理对象生命周期，更安全灵活。
