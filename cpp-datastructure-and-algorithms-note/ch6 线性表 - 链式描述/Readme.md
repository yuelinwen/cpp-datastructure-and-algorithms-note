ch6 线性表 链式描述

元素地址随机分布

数据对象实例的每一个元素都用一个单元或节点来描述
每一个节点包含链域 + 数据域。链域指的是下一个结点的pointer信息

记住精华： 插入删除只改指针，访问查找要遍历。


| 操作             | 最好情况      | 最坏情况 | 说明          |
| -------------- | --------- | ---- | ----------- |
| 按位置访问 `get(i)` | O(1)（i=1） | O(n) | 需从头顺序查找     |
| 按值查找 `find(x)` | O(1)      | O(n) | 顺序查找        |
| 插入（已知前驱）       | O(1)      | O(1) | 只改指针        |
| 插入（按位置 i）      | O(1)（i=1） | O(n) | 先找到位置       |
| 删除（已知结点）       | O(1)      | O(1) | 指针重连        |
| 删除（按位置 i）      | O(1)（i=1） | O(n) | 先找到前驱       |
| 求长度 `length()` | O(n)      | O(n) | 需遍历（若未维护计数） |

删除标准流程：
找到 前驱结点 prev
保存要删的结点：Node* target = prev->next;
让 prev 指向 target->next   prev->next = target->next;
删除结点   delete target;







类型		一句话描述
单向链表		每个结点只有 next，最后指向 nullptr
单向循环链表	最后一个结点 next 指回头结点
双向链表		每个结点有 prev 和 next
双向循环链表	头尾相连的双向链表
