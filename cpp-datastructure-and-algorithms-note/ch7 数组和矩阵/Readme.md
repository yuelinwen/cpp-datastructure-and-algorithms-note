ch 7 矩阵


| 分类 | 矩阵名称  | 是否方阵 | 主要特征（一句话）     |
| -- | ----- | ---- | ------------- |
| 形状 | 行矩阵   | ❌    | 只有一行（1×n）     |
| 形状 | 列矩阵   | ❌    | 只有一列（n×1）     |
| 形状 | 矩形矩阵  | ❌    | 行数 ≠ 列数       |
| 形状 | 方阵    | ✅    | 行数 = 列数       |
| 元素 | 零矩阵   | 视情况  | 所有元素为 0       |
| 元素 | 对角矩阵  | ✅    | 非主对角线全为 0     |
| 元素 | 标量矩阵  | ✅    | 对角线元素相同       |
| 元素 | 单位矩阵  | ✅    | 主对角线为 1，其余为 0 |
| 元素 | 上三角矩阵 | ✅    | 主对角线下方全为 0    |
| 元素 | 下三角矩阵 | ✅    | 主对角线上方全为 0    |
| 性质 | 对称矩阵  | ✅    | A = Aᵀ        |
| 性质 | 反对称矩阵 | ✅    | Aᵀ = −A       |
| 性质 | 正交矩阵  | ✅    | AᵀA = I       |
| 性质 | 可逆矩阵  | ✅    | 存在逆矩阵 A⁻¹     |
| 性质 | 奇异矩阵  | ✅    | 不可逆           |
| 存储 | 稀疏矩阵  | 视情况  | 非零元素很少        |
| 存储 | 稠密矩阵  | 视情况  | 非零元素很多        |
| 应用 | 邻接矩阵  | ✅    | 图的边关系表示       |
| 运算 | 转置矩阵  | 视情况  | 行列互换          |
