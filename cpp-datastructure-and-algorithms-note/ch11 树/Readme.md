ch11 树


二叉树
由有限个结点组成的集合，或者为空；若不为空，则由一个根结点及其左、右两个互不相交的二叉子树组成。
二叉树的基本特点
每个结点最多两个孩子
不是必须有两个
可以有 0、1 或 2 个
有左右之分
左子树和右子树是有顺序的
左右交换，结构就变了
可以是空树 根结点都没有


二叉树四种遍历方法：前序遍历，中序遍历，后序遍历，层次遍历


二叉搜索树（BST）
二叉搜索树是二叉树的一种
BST 额外规则：
左子树所有结点 < 根
右子树所有结点 > 根


常见的二叉树类型（常考）
1️⃣ 满二叉树
每一层结点数都达到最大
结点数：2^h - 1
2️⃣ 完全二叉树
除最后一层外全满
最后一层从左往右连续
3️⃣ 二叉搜索树（BST）
左 < 根 < 右
4️⃣ 平衡二叉树
左右子树高度差 ≤ 1
